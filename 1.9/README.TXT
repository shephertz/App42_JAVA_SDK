//App42 Java SDK

1. UnZip the downloaded file

2. This will contain App42_SERVICE_JAVA_API-X.X.X.jar, lib, docs, sample folder and README.txt

3. lib folder contains all the third party jar files used by the App42 Cloud Java SDK.

4. docs folder contains JAVA docs

5. Sample folder contains eclipse sample project for using App42 SERVICE JAVA SDK.

6. You need to put App42_SERVICE_JAVA_API-X.X.X.jar in classpath of Java Project to use this.

7. Please visit http://api.shephertz.com/cloudapidocs/index.php for detail documentaion.
